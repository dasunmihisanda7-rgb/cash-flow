# Cash Flow - Financial Dashboard

A modern, high-performance financial tracking application built with Next.js, Framer Motion, and Supabase.

## Features
- **Income & Expense Tracking**: Manage your finances with ease.
- **Dynamic Visuals**: interactive charts using Recharts.
- **Micro-animations**: Smooth transitions using Framer Motion.
- **Real-time Data**: Powered by Supabase.

## Deployment Status
The project is ready for deployment. All local changes have been synced to GitHub.

- **GitHub Repository**: [dasunmihisanda7-rgb/cash-flow](https://github.com/dasunmihisanda7-rgb/cash-flow)

## Deployment Instructions (Vercel)

1. **Connect GitHub**: Import the project into Vercel from your GitHub account.
2. **Environment Variables**: You must add the following variables in the Vercel dashboard:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. **Build Settings**: The default Next.js settings will work out of the box.

---

## Local Development

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
